## Remote Control App
This app allows you to control your computer from another device.
App works in system tray.

### Supported OS:
Windows

### Features
Keyboard and mouse control
- mouse move
- mouse click
- key click
- keyboard shortcuts (remoteserver.SystemControl.Shortcut enum)
- media keys (next, previous, volumeUp, etc.)

System actions
- hibernate
- restart (with or without timeout)
- logout
- abort system shutdown
- shutdown (with or without timeout)
- windows lock
